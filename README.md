## 🤔 요구사항 분석 및 피쳐 리스트업

<details>
<summary>피그마를 통한 피쳐 리스트업</summary>

### 메인페이지

- 상단 배너 광고 리스트 조회
    - 배너 이미지, 외부 광고 링크
- 인기 동아리 모집 게시글 리스트 조회
    - 동아리 모집 게시글 첨부 사진, 모집 게시글 제목, 조회수, 댓글수
- 인기 대외활동 모집 게시글 리스트 조회
    - 대외활동 모집 게시글 첨부 사진, 모집 게시글 제목, 조회수, 댓글수
- 인기 공모전 모집 게시글 리스트 조회
    - 공모전 모집 게시글 첨부 사진, 모집 게시글 제목, 조회수, 댓글수
- 이벤트 정보 리스트 조회
    - 이벤트 첨부 사진, 이벤트 정보 이름, 이벤트 외부 링크

### 동아리 정보 조회 페이지

- 동아리 카테고리 리스트 조회
    - 카테고리 이름
- 인기 동아리 모집 공고 리스트 조회
    - 동아리 모집 게시글 첨부 사진, 모집 게시글 제목, 조회수, 마감 디데이
- 동아리 랭킹 리스트 조회
    - 동아리 이름, 동아리 소개 정보, 동아리 프로필 사진
- 동아리 탐색
    - 동아리 이름, 동아리 프로필 사진, 게시글 첨부 사진, 모집 게시글 제목, 조회수, 댓글수, 마감 디데이
    - 카테고리, 동아리 활동 지역, 마감된 공고 여부로 필터링 필요, 정렬 기준 적용
    - 페이지네이션 적용

### 동아리 정보 검색

- 키워드로 검색
    - 키워드는 동아리 이름, 동아리 모집 게시글 제목에 적용
- 동아리 카테고리로 필터링
    - 전체, 문화/예술/공연, 봉사/사회활동, 학술/교양, 취업/창업, 어학, 체육, 친목, 기타
- 마감 기한으로 필터링
    - 전체, 7일 이하, 7일 이상, 14일 이상, 30일 이상
- 활동 지역으로 필터링
    - 전국, 수도권, 충북/충남/대전, 전북, 전남/광주, 경북/대구, 경남/부산/울산, 강원, 제주, 기타
- 활동 요일로 필터링
    - 전체, 월요일, 화요일, 수요일, 목요일, 금요일, 주말

### 동아리 정보 검색 후 정렬

- 최신순
- 인기순
- 관련도순
- 거리순

### 메인페이지

- 상단 배너 광고 리스트 조회

    - 5개씩 전달, 인기 동아리 모집글의 기준은 조회수 높은 순
- 인기 대외활동 모집 게시글 리스트 조회
    - 5개씩 전달, 인기 대외활동 모집글의 기준은 조회수 높은 순
- 인기 공모전 모집 게시글 리스트 조회
    - 5개씩 전달, 인기 공모전 모집글의 기준은 조회수 높은 순

### 동아리 정보 조회 페이지

- 인기 동아리 모집 공고 리스트 조회
    - 5개씩 전달, 인기 동아리 모집글의 기준은 조회수 높은 순
- 동아리 랭킹 리스트 조회
    - 3개씩 전달, 인기 동아리의 기준은 좋아요수 높은 순,
- 동아리 탐색
    - 동아리 모집 게시글 표시
    - 10개씩 전달, 기본적으로는 생성일자 최신 순으로 정렬

### 동아리 정보 검색

- 키워드는 동아리 이름, 동아리 모집 게시글 제목에 적용
- 동아리 카테고리, 마감 기한, 활동 요일, 활동 지역의 필터 요소는 중복 선택 안되고 한 개씩만 선택됨

## ✨ 최종 구현 설계 및 공수 분담

<details>
<summary>구현 기능 선정 및 설계</summary>

### 최종 구현 기능

- 인기 동아리 모집 게시글 리스트 조회
    - 동아리 모집 게시글 첨부 사진, 모집 게시글 제목, 조회수, 댓글구, 마감 디데이 등의 데이터를 포함
    - 조회수 높은 순으로 정렬하여 상위 5개의 결과만 전달
- 인기 대외활동 모집 게시글 리스트 조회
    - 대외활동 모집 게시글 첨부 사진, 모집 게시글 제목, 조회수, 댓글수 등의 데이터를 포함
    - 조회수 높은 순으로 정렬하여 상위 5개의 결과만 전달
- 동아리 랭킹 리스트 조회
    - 동아리 이름, 동아리 소개 정보, 동아리 프로필 사진 등의 데이터를 포함
    - 좋아요수 높은 순으로 정렬하여 상위 3개의 결과만 전달
- 동아리 정보 검색
    - 키워드로 동아리 이름, 동아리 모집 게시글 제목에 대한 검색
    - 동아리 카테고리, 마감 기한, 활동 요일, 활동 지역의 필터 요소를 적용하여 동아리 모집글을 검색
    - 기본 정렬은 작성일 최신순이며 페이지네이션은 배제
